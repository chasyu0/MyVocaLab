# MyVocaLab

📘 **다국어 단어 학습 앱 (React Native) 개발 실험 + 개인 스터디 프로젝트**

---

## 📌 프로젝트 성격

> 이 프로젝트는 **학습(80%) + 개발 실험(20%)** 목적의 **개인 스터디용 프로젝트**입니다.  
> **모든 기능 구현은 “시도 중심”이며, 완성/배포를 보장하지 않습니다.**  
> 진행 방식 및 속도는 학습 상황과 일정에 따라 조정됩니다.

---

## 🛠️ 기술 및 학습 스코프

- **React Native (초학습 단계)** → 개인 앱 개발 실험
- **Frontend (React 경험 기반)** → 모바일 UI로 확장 도전
- **Language**: 다국어 UI 구성 가능성 있는 MVP 기반 설계
- **기능**: 단어 목록 생성 / 학습 인터페이스 설계 / 이후 확장

> ⚠ 네이티브 앱은 아직 학습 중으로, 실제 개발 전 **관련 스터디를 병행**합니다.  
> ⚠ 기능/아이디어 일부는 초기에는 비공개 또는 미구현 상태일 수 있습니다.

---

## 📅 진행 방식 및 규칙

- **학습·연습 기반**: 기능 구현 단계별 실험/검토 → 괜찮으면 개발
- **진행 속도**: **취준 + 알바 + 스터디 병행으로 인해 불규칙 (느리거나 중단될 수 있음)**
- **베이퍼웨어 방지용**: 초기 단계에서 scope 조절
- **규칙**:
  - 계획은 엑셀로 관리 (내부 문서)
  - 기능은 단계별로 MVP 기준보다 확장하지 않음
  - 중간중간 리드미 및 로그만 업데이트
  - 필요 시 기능 및 구조 변경 가능

---

## 🤝 협업 관련

본 프로젝트는 **현재 1인 개발 및 학습 중심**으로 운영 중입니다.  
아래 조건에 해당할 경우 협업 검토 가능합니다.

- 코드 리뷰 및 피드백이 가능한 실력자 (알고리즘/자료구조 이해 포함)
- 앱 개발 관련 실무 또는 배포 경력자 우대
- 자기 학습 + 기술 탐구 기반의 사이드 프로젝트를 선호하는 분

**→ 단, 상세 기획/데이터 구조 등 핵심 정보는 논의 후 일부 공개됩니다.**
